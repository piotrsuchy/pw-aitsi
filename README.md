# PW - AiTSI - project

There are three markdown files with documentation:

- project-requirements.md - functional and non-functional
- project-verification.md - describes how different functionalities should be verified
- criteria.md - describes how the project is going to be graded

## Web - getting started

First, run the development server:

```bash
cd web/
npm run dev
```

```bash
cd web/
bun dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
